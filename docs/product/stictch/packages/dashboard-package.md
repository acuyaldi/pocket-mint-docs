---

# Product Identity

Pocket Mint is a Private Financial Workspace.

It helps users answer:

• What do I own?

• What do I owe?

• What requires attention?

• What happened recently?

The interface should feel:

- Calm
- Professional
- Trustworthy
- Information-first
- Privacy-first

Avoid:

- Expense Tracker
- Banking Dashboard
- Crypto Dashboard
- Analytics Platform
- Marketing Website

---

# Screen Goal

The Dashboard provides an immediate overview of the user's financial position.

Users should understand their situation within five seconds.

---

# Primary User Questions

What is my financial position today?

Where is my money?
