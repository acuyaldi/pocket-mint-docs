# Purpose

This document defines the visual composition of the Pocket Mint Landing page.

Unlike authenticated screens, the Landing page should establish trust before introducing features.

The experience should feel calm, structured, and intentional.

Visitors should understand Pocket Mint before deciding whether to use it.

---

# Composition Philosophy

The Landing page is not a marketing campaign.

It is a product introduction.

Every section should answer one visitor question.

Avoid exaggerated storytelling.

Avoid overwhelming visitors with feature lists.

Trust should grow naturally as users scroll.
