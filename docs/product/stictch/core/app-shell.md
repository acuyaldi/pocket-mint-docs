# App Shell Design Brief

> Pocket Mint v2

---

# Purpose

The App Shell provides consistent navigation throughout Pocket Mint.

Users should always know:

- where they are
- where they can go
- what actions are available

---

# Navigation

Primary Navigation

- Dashboard
- Wallets
- Transactions
- Installments
- Reports
- Settings

---

# Desktop

Persistent sidebar.

Main content centered.

Large working canvas.

---

# Tablet

Collapsible sidebar.

---

# Mobile

Bottom navigation.

Overflow menu for secondary pages.

---

# Header

Contains:

- page title
- global search (future)
- notifications (future)
- profile

---

# Footer

None.

Avoid unnecessary footer content.

---

# Principles

Navigation should disappear into the background.

The user's attention belongs to financial information.

---

# Never

Never make navigation visually louder than page content.

Never duplicate navigation.

Never hide essential pages behind multiple clicks.

---

# Stitch Notes

The App Shell should feel invisible.

Navigation supports work.

It is not the focus.
